# Automobile Sales Analysis & Dashboard

This project analyzes automobile sales data using Python and Power BI.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Power BI

## Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis
- Sales Trend Analysis
- State-wise Demand Analysis
- Vehicle Segment Market Share
- Interactive Power BI Dashboard

## Files
- sales.py → Python analysis script
- automobile_data.csv → Dataset
- PowerBI_Dashboard.pbix → Power BI dashboard
- dashboard_screenshot.png → Dashboard preview
